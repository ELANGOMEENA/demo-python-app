# Welcome Page Python App

A simple Python Flask application that serves a beautiful **DevOps Welcome Page**.  
This app demonstrates how to containerize a Python web application with Docker and serve an HTML page.

---

## Features

- Serves a modern HTML welcome page
- Fully containerized with Docker
- Easy to deploy locally or on cloud platforms

---

The Docker image is publicly available on Docker Hub:  

**URL:** [https://hub.docker.com/r/satishkarengula/demo-pyapp](https://hub.docker.com/r/satishkarengula/demo-pyapp)

You can pull it directly using:

```bash
docker pull satishkarengula/demo-pyapp:v1
```
